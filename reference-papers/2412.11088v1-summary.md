# Seeing the Forest and the Trees: Solving Visual Graph and Tree Based Data Structure Problems Using Large Multimodal Models

## Summary

This paper asks a simple but important question: **can large multimodal models solve graph and tree data-structure problems from images alone?** The answer is **yes, to a meaningful extent**, especially for trees, and that creates a real assessment problem for computing education.  

## What the paper does

The authors build a **new benchmark dataset of 9,072 image-based tasks** covering:

* binary trees
* binary search trees
* undirected graphs
* directed graphs

They test several multimodal models, including **GPT-4o, GPT-4V, Gemini 1.5 Pro, Gemini 1.5 Flash, Gemini 1.0 Pro Vision, and Claude 3 models**. The dataset was generated programmatically to reduce data leakage, so the problems were not just scraped from the internet.  

## How they tested it

Each task paired:

* an **image** of a graph or tree
* a **text prompt** asking for an operation such as:

  * traversal
  * adjacency list
  * breadth-first search
  * depth-first search

The prompting was intentionally **simple zero-shot prompting**, with no step-by-step hints. The goal was to test what a student might realistically do with minimal effort. Accuracy was measured using **pass@1** and **pass@3**. 

## Main findings

### 1. Trees were much easier for the models than graphs

The strongest result was on tree problems.

* **GPT-4o** was best on trees with **87.6% pass@3**
* On graphs, the best overall result came from **Gemini 1.5 Flash** with **56.2% pass@3**
* GPT-4o dropped to **44.7% pass@3** on graphs

So these models are already quite capable on visual data-structure questions, but their performance is much stronger on trees than on general graphs.  

### 2. Structural complexity mattered more than visual styling

The paper’s strongest analytical result is that **structural features** mattered more than aesthetic ones.

The biggest drivers of worse performance were:

* more edges
* more nodes
* higher density
* more complex degree distributions
* harder layouts

Accuracy generally **fell as the number of edges increased**. The chart on page 7 shows this clearly across models. 

### 3. Aesthetic features mattered less

Things like:

* edge width
* node color

had **smaller and less consistent effects**. One exception was that **white nodes on a white background** may hurt performance more than higher-contrast colors like yellow. But overall, visual styling was less important than the underlying structure. 

### 4. Binary search trees were the easiest

Models did best on **binary search trees**, likely because they have more regular structure and more predictable spatial layout. The authors suggest that this consistent top-down organization helps vision models parse them more reliably than arbitrary graph layouts. 

## Why this matters

The paper’s core implication is not just technical. It is educational.

A common idea in computing education has been: “use diagrams or visual questions to make AI cheating harder.” This paper argues that this defense is weakening fast. If multimodal models can already solve many visual tree and graph problems, then **image-based assessments are not a durable workaround**. 

The authors push a stronger claim: educators may need to rethink assessment altogether, moving away from traditional question formats and toward approaches that focus more on:

* deeper understanding
* project-based work
* critical engagement with AI tools

They also note a positive side: these models could help students learn difficult data-structure topics, especially graphs and trees, which many students struggle with. 

## Limitations

The paper is useful, but not broad enough to overstate.

Main limits:

* only a subset of data structures was tested
* only core operations were included
* only a limited set of visual variations was studied
* prompting was basic zero-shot only
* model settings were mostly left at default
* model capabilities are changing quickly, so results may age fast

So the paper gives a strong snapshot, not a final answer. 

## Bottom line

This paper shows that **large multimodal models already have non-trivial spatial reasoning ability on visual data-structure problems**. They are especially strong on trees, weaker but still capable on graphs, and their errors are driven more by **structural complexity** than by superficial image style. For computing education, that means visual assessments are no longer a safe assumption for AI resistance.  

## One-sentence version

**The paper shows that modern multimodal models can solve many image-based graph and tree problems, especially trees, which weakens the idea that visual CS assessments are resistant to AI.**
